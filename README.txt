# Plomberielise — Site vitrine

Site one-page professionnel pour Plomberielise, plombier basé à Fort-de-France (Martinique).

---

## Structure

```
site-vitrine/
├── index.html        # Page principale (HTML)
├── css/
│   └── style.css     # Tous les styles
├── js/
│   └── main.js       # Scripts (navbar, animations, modal)
└── assets/
    ├── fonts/
    ├── icons/
    └── img/
```

---

## Stack

- HTML5 / CSS3 / JavaScript vanilla
- Google Fonts — Inter
- Font Awesome 6.5 (CDN)
- Aucune dépendance NPM, aucun build requis

---

## Sections

1. Hero — Titre, slogan, CTA téléphone + WhatsApp, note Google
2. Services — 12 prestations en 3 catégories (Détection, Installation, Réparation)
3. Stats — Chiffres clés (24h, 60+ avis, devis gratuit)
4. Pourquoi nous — 4 arguments différenciants
5. Zone d'intervention — 22 communes en Martinique
6. Avis clients — 3 vrais avis Google / Pages Jaunes
7. Contact — CTA téléphone, WhatsApp, adresse
8. Footer — Nav + mentions légales (modal)

---

## Déploiement (Vercel)

1. Connecter le repo GitHub à Vercel
2. Framework preset : **Other**
3. Output directory : `.` (racine)
4. Aucune variable d'environnement requise
5. Deploy
